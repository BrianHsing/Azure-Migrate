# Azure-Migrate-Hand-on-Lab
 透過Azure VM巢狀虛擬化建立單一Hyper-V主機，利用Azure Migrate輕鬆地將虛擬機器搬遷至Azure

<h1>環境準備</h1><br>
 √Azure訂用帳戶、Azure Subscription owner role<br>
 √使用Single-Hyper-V.ps1佈署Hyper-V Server<br> (21分鐘)
 √設定客體虛擬機器的網際網路連線<br> (5分鐘)
 √匯入Linux Server<br>
 √匯入Windows Server<br>

<h1>遷移流程</h1><br>
 √建立Azure Migrate<br>
 √準備評估Hyper-V VM<br>
 √準備遷移Hyper-V VM<br>
